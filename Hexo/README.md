# Hexo

# Installation

## Install nvm

```bash
wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh
```

### Install stable node

```bash
nvm install stable
```

## Install nodejs-legacy

```bash
sudo apt-get install -y nodejs-legacy
```

## Install npm

```bash
sudo apt-get install -y npm
```


## Install hexo

```bash
sudo npm install hexo-cli -g
```
